# Ray Tracer
Ray Tracer built in C++ for CS130

<br>
Dependencies:
<br>
- Scons (to build executable)
<br>
	- `apt-get install scons`
<br>
- libpng-dev
<br>
	- `sudo apt-get install libpng-dev`
<br>
<strong>Run Instructions: </strong>
<br>
`$ cd proj-rt-files`
<br>
`$ ./ray_tracer -i <test_file.txt>`
